# AutoGo PPOCRv5 Socket OCR（中文说明文档）

一个基于 Lua + TCP Socket 的 OCR 自动化模块，用于与本地 OCR
服务通信，实现屏幕文字识别、找字定位与自动点击功能。

------------------------------------------------------------------------

# 🚀 功能特点

-   📡 基于 TCP Socket 与 OCR 服务通信\
-   🔍 支持全屏文字识别\
-   🎯 支持找字定位（中心 / 随机点击）\
-   🖱️ 支持自动点击识别结果\
-   ⚡ JSON 自动转义处理\
-   🧠 支持多架构（arm64 / x86 / x86_64）

------------------------------------------------------------------------

# 📦 运行环境

## 必需环境

-   Lua 解释器（支持 socket）
-   LuaSocket（require("socket")）
-   Android root 或 shell 执行权限
-   对应架构的 OCR 二进制文件

# ⚙️ OCR启动流程

## 1️⃣ 获取设备架构

exec("getprop ro.product.cpu.abi")

支持： - arm64-v8a - x86 - x86_64

------------------------------------------------------------------------

## 2️⃣ 启动OCR服务

pkill -f xxx chmod 755 /data/local/tmp/xxx nohup xxx 9091 \> /dev/null
2\>&1 &

默认端口：9091

------------------------------------------------------------------------

## 3️⃣ 建立Socket连接

local socket = require("socket") local client = socket.tcp()
client:connect("127.0.0.1", 9091)

------------------------------------------------------------------------

# 🔥 API接口说明

------------------------------------------------------------------------

## 📌 1. 全屏识别

ScreenOCR(x1, y1, x2, y2, color)

示例：

local result = ScreenOCR(0, 0, 1080, 1920) print(result)

------------------------------------------------------------------------

## 📌 2. 找字（返回区域）

FindTextRegion(x1, y1, x2, y2, text, color)

返回： { true, x, y, width, height }

------------------------------------------------------------------------

## 📌 3. 随机点击找字

FindTextRandomClick(x1, y1, x2, y2, text, click, color)

------------------------------------------------------------------------

## 📌 4. 中心点击找字

FindTextCenterClick(x1, y1, x2, y2, text, click, color)

------------------------------------------------------------------------

# 🧠 JSON安全处理

-    → \\
-   " → \"\
-   换行 → `\n  `{=tex}
-   回车 → `\r  `{=tex}

------------------------------------------------------------------------

# 📡 通信协议

请求格式：

{ "type": "all/find/rand/center", "x1": 0, "y1": 0, "x2": 100, "y2":
100, "text": "xxx", "colorStr": "" }

------------------------------------------------------------------------

# 📡 返回格式

坐标返回：

{ "x": 100, "y": 200 }

区域返回：

{ "X": 10, "Y": 20, "Width": 100, "Height": 50 }

------------------------------------------------------------------------

# ⚠️ 注意事项

-   OCR服务必须先启动\
-   默认端口 9091\
-   需要可执行权限\
-   socket失败返回nil

------------------------------------------------------------------------
